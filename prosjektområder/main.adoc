== asdsad
Prosjektområder opprettes normalt som egne GitHub repositories med skriverettigheter kun for prosjektmedlemmene. Andre kan bidra ved å lage "fork" og "pull request". Leveranser i form av dokumentasjon kopieres til "riktig" sted i strukturen under Nasjonalt arkitekturbiblitotek før prosjektslutt, mens arbeidsversjoner, møtereferater og annet prosjektspesifikt innhold tas vare på i prosjektområdet så lenge det er interessant.
